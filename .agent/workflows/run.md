description: How to run the Growth Map Portal locally with Vite
---
// turbo-all
1. Install dependencies: `npm install`
2. Start the development server: `npm run dev`
3. Open the browser to the URL shown in the terminal (usually `http://localhost:5173`)
4. Ensure you permit microphone access when prompted.
5. Tip: For the best experience, use Chrome or Edge for the most reliable Speech Recognition API.
